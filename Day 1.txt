HTML Best Practices

-Use lowercase when writing html
-Try to keep your html file small
-Use _blank for external links only and always (opens in a new window/tab)
-Tackle accessibility, SEO, and performance
-Remove comments from production code
-Use HTML entities to replace unsafe characters:
	&nbsp (non-breaking space); &copy (copyright symbol); &lt (<); &gt (>); &amp (&);
